# entreniebla

**entreniebla** is a limited-edition book published in 2014, exploring algorithmic visual poetry and codeworks. Each piece is presented as a captured accident of process in motion, alongside the code that generated it.

The book traces routes between poetic expression and programming languages. Some pieces use indentation, comments, and syntax to produce meaning beyond what the code executes. Others build and deconstruct words through algorithmically generated geometric forms. How does poetic meaning emerge from the interaction between code and generative systems? Meaning does not live in the parts, it emerges from gesture, rhythm, and improvisation between human, machine, and language.

In 2026 the book left paper behind to live on the web. Every piece runs live in the moment you look at it, generating something different each time, existing only while a browser is open, then disappearing.

## About this project

A browser-based reinterpretation of a print artifact, rebuilding static poetry as generative, real-time code. Each piece is a self-contained sketch (JavaScript / p5.js) exploring procedural typography, algorithmic form generation, and code-as-poetic-material.
